丰富的项目经验 不断学习并付诸实践

  • 无尽试炼

    无尽试炼是基于 egret 游戏引擎开发的一款H5挂机类游戏，主要涉及的技术细节有：

    • 结合 redux 管理游戏前端的数据，避免手动更新每个页面的数据。

    • 利用 Promise 对象 包装动画及公共动效，代码变得清晰易懂，降低了代码耦合度。

    • 组件化模式开发。

    • 接入了 微信 JSSDK 以及多个第三方渠道。

  • 游戏管理后台

    本项目还在不断地迭代，目前的主要功能有用户系统/对账系统/游戏数据分析系统等等。主要涉及的技术细节有：

    • 前端基于 react 开发，使用了 react-router / antd design 等模块。

    • 用户登陆验证基于 JWT （json web token）

    • 项目通过 webpack 进行打包构建。
